Ruby 3.0 is here, optimized for iOS 26, macOS Tahoe, and Liquid Glass.

What's new:

- All new interface with an emphasis on images
- Article summaries have finally arrived, long press to see it in action (Apple Intelligence device required)
- Trending has been replaced with a comprehensive Digest feed, allowing you to easily read a comprehensive breakdown of what's going on in each topic
- All new in-house API with much improved sourcing
- Enhanced notifications
- Fixed Apple Watch app

Ruby is available on the App Store for $6.99.
